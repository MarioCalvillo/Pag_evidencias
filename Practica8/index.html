Proyecto de Ing.

Departamento de Ciencias e Ingenierías | Universidad Iberoamericana Puebla, México.

Automata

Mario Alberto Calvillo Ceron
Ing. Mecatronica
Fecha: 16/08/24

- Resumen -

Este proyecto describe la creación de un autómata mecánico de madera, fabricado con MDF cortado a láser, que simula una banda de camarones (cinco en total) tocando instrumentos y moviéndose. Las piezas móviles fueron ensambladas de manera manual, sin el uso de componentes eléctricos, lo que permite que los camarones se muevan de forma mecánica mediante un sistema de engranajes y ejes.

- Introducción -

La construcción de autómatas mecánicos es una fascinante combinación de arte y mecánica. En este proyecto, se diseñó un autómata sin componentes eléctricos, en el que una banda de camarones (cada uno tocando su propio instrumento) cobra vida mediante un sistema de movimiento mecánico. El uso de corte láser para las piezas de MDF permitió una precisión y facilidad en el ensamblaje, logrando que las piezas móviles se coordinasen para dar la ilusión de un concierto.

- Materiales -

  • MDF: Tablero de fibra de densidad media utilizado como material principal para las piezas del autómata.
  • Engranajes y ejes: Componentes para el sistema de movimiento del autómata.
  • Cinta adhesiva o pegamento fuerte: Herramientas para fijar las piezas durante el ensamblaje.
  • Herramientas de corte láser: Utilizadas para cortar las piezas de MDF con precisión.
  • Herramientas para ensamblaje de piezas mecánicas: Incluyen destornilladores y otras herramientas para unir los componentes mecánicos del autómata.
  • Pintura y decoración: Materiales utilizados para pintar y decorar los camarones e instrumentos del autómata.

- Desarrollo -

  • Diseño de las piezas: Se diseñaron las piezas para los camarones, los instrumentos y el sistema de engranajes utilizando un software CAD, asegurando que las partes encajaran perfectamente.
  • Corte de piezas: Con la máquina de corte láser, se cortaron las piezas de MDF según el diseño, prestando atención a la precisión para que las partes móviles pudieran moverse sin dificultad.
  • Montaje del sistema de movimiento: Se ensamblaron los engranajes y ejes que permitieron el movimiento de las piezas. Estos mecanismos transformaron el movimiento manual en movimiento de las piezas de los camarones.
  • Ensambaje de las piezas: Las piezas cortadas se ensamblaron en la estructura principal, asegurando que el sistema de movimiento mecánico funcionara de manera eficiente para que los camarones se movieran de acuerdo con la acción deseada.
  • Pruebas de movimiento: Se realizaron pruebas para asegurar que los camarones se movieran adecuadamente, tocando sus instrumentos de manera coordinada.

Diseño

El diseño incluye una banda de camarones, cada uno con un instrumento musical y piezas articuladas que permiten el movimiento. El sistema de engranajes y ejes fue cuidadosamente diseñado para que cada camarón se moviera de manera sincronizada con el resto de la banda, creando una actuación mecánica realista.

Construcción

La construcción se centró en el corte preciso de las piezas con láser y el ensamblaje del sistema mecánico que permitiera el movimiento de las piezas sin necesidad de electricidad. El proceso incluyó la instalación de los engranajes y ejes para garantizar que las piezas móviles estuvieran sincronizadas.

- Resultados -

El autómata mecánico fue completado con éxito, logrando que los camarones se movieran y tocaran sus instrumentos de manera sincronizada. El resultado fue una pieza funcional que mostró cómo se puede lograr un movimiento dinámico sin el uso de electricidad, solo con mecanismos mecánicos.

- Conclusiones -

Este proyecto demostró que, con la precisión del corte láser y la planificación adecuada del sistema mecánico, es posible crear un autómata funcional sin necesidad de componentes eléctricos. La interacción entre los engranajes y las piezas móviles logró una representación efectiva y entretenida de una banda de camarones tocando música.

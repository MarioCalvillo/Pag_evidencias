Departamento de Ciencias e Ingenierías | Universidad Iberoamericana Puebla, México.

Impresion 3D

Mario Alberto Calvillo Ceron
Ing. Mecatronica
Fecha: 24/08/24

- Resumen -

En esta práctica, el estudiante diseña y fabrica un modelo articulado de un muñeco de jengibre mediante el uso de SolidWorks y una impresora 3D. El objetivo es aprender el proceso de diseño en 3D y la preparación de modelos articulados para impresión, explorando las capacidades de fabricación aditiva y las limitaciones de los materiales de impresión 3D.

- Introducción -

La impresión 3D es una tecnología que permite la creación de objetos físicos a partir de modelos digitales, lo que facilita el prototipado rápido y la producción de piezas personalizadas. En esta práctica, el estudiante diseña un muñeco de jengibre articulado en SolidWorks y lo fabrica mediante una impresora 3D. Esta actividad permite desarrollar habilidades en diseño 3D, familiarizarse con el software de modelado y aprender sobre los aspectos técnicos de la impresión 3D.

- Materiales -

  • SolidWorks: Software de diseño CAD usado para crear el modelo 3D del muñeco de jengibre.
  • Impresora 3D: Dispositivo que fabrica el modelo mediante la tecnología de fabricación aditiva.
  • Filamento PLA (o material de impresión compatible): Material plástico utilizado en la impresora para crear el modelo.

- Desarrollo -

  1. Diseño en SolidWorks:
    • Se crea un modelo articulado del muñeco de jengibre en SolidWorks, utilizando herramientas de modelado 3D y ensamblaje para diseñar las articulaciones y definir las partes móviles.
    • El diseño se verifica en SolidWorks para asegurar que las piezas encajen correctamente y puedan moverse de forma articulada.
  2. Preparación para la impresión 3D:
    • El modelo se exporta a un formato compatible para impresión (como .STL).
    • En el software de la impresora, se ajustan parámetros como la resolución de impresión, la densidad del relleno y el tipo de soportes necesarios para las partes articuladas.
  3. Impresión 3D:
    • Se prepara la impresora con el filamento adecuado (PLA) y se inicia el proceso de impresión.
    • El modelo se imprime por capas, y se supervisa el proceso para asegurar que las piezas articuladas se impriman correctamente.

Diseño

El diseño del muñeco de jengibre articula sus extremidades para permitir movimientos en las piernas y brazos. Se toman en cuenta detalles como la proporción y el grosor de las piezas para facilitar la articulación sin comprometer la estabilidad del modelo. Además, se diseñan detalles decorativos para darle un aspecto auténtico de muñeco de jengibre.

- Resultados -

El resultado final es un muñeco de jengibre articulado que puede moverse en las articulaciones y mantiene la estética de un muñeco tradicional. A través de esta actividad, el estudiante adquirió habilidades en diseño 3D y en la preparación de modelos articulados para impresión, explorando las posibilidades de la tecnología de fabricación aditiva.

- Conclusiones -

La práctica permitió al estudiante llevar a cabo el proceso completo de diseño y fabricación de un modelo articulado mediante impresión 3D. Además de mejorar las habilidades en el manejo de SolidWorks y en la configuración de impresión, se profundizó en el conocimiento sobre las limitaciones y posibilidades de los materiales de impresión 3D. Esta experiencia es útil para futuros proyectos de prototipado y diseño de modelos complejos.

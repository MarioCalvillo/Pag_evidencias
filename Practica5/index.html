Molde de silicon

Mario Alberto Calvillo Ceron
Ing. Mecatronica
Fecha: 14/09/24

- Resumen -

En este proyecto se detalla la creación de un molde de silicón, utilizando un contramolde elaborado previamente en la Práctica 4. El diseño central está inspirado en un modelo de Batman, destacando técnicas de precisión para obtener un resultado fiel al original.

- Introducción -

La fabricación de moldes es una técnica esencial en el diseño y la producción, ya que permite la reproducción precisa de formas complejas. En este caso, se empleó silicón debido a su flexibilidad y resistencia, asegurando la durabilidad del molde. A partir de un contramolde creado en una práctica anterior, este proyecto busca demostrar los pasos clave para elaborar un molde funcional con acabados de alta calidad.

- Materiales -

  • Silicón para moldes: Material utilizado para crear la base del molde.
  • Contramolde: El molde original usado para crear el diseño del contramolde de Batman.
  • Catalizador: Componente que acelera el proceso de curado del silicón.
  • Recipiente para mezclar: Utensilio utilizado para mezclar los ingredientes del silicón y el catalizador.
  • Espátula para mezclar: Herramienta para mezclar los componentes de forma eficiente.

- Desarrollo -

  • Preparación del Contramolde: Revisar el contramolde en busca de imperfecciones y limpiar cualquier residuo que pueda afectar el molde.
  • Mezcla del Silicón: Combinar el silicón con el catalizador siguiendo las proporciones indicadas por el fabricante.
  • Vaciado del Silicón: Verter la mezcla de silicón de manera uniforme sobre el contramolde, evitando la formación de burbujas de aire.
  • Secado: Dejar reposar el molde durante el tiempo necesario para que el silicón cure completamente.

Diseño

El diseño elegido es un modelo de Batman, conocido por sus detalles icónicos. El proceso busca preservar las características principales del diseño para garantizar un acabado profesional.

- Resultados -

El molde final fue capaz de reproducir con precisión los detalles del contramolde, mostrando una alta fidelidad al diseño original. Esto permite múltiples reproducciones sin perder calidad ni detalles importantes.

- Conclusiones -

Este proyecto destacó la importancia de una preparación adecuada del contramolde y una mezcla precisa del silicón. El uso de materiales de calidad y técnicas adecuadas garantizó un resultado profesional, ideal para futuras aplicaciones en proyectos de diseño y producción.

Definición de Página Web

La Real Academia Española menciona que una página web es un conjunto de informaciones de un sitio web que se muestran en una pantalla y que puede incluir textos, contenidos audiovisuales y enlaces con otras páginas. Dando como referencia una manera de usar la tecnología con el fin de brindar información al público respecto a promocionar alguna marca, libros, sitios académicos, restaurantes, locales, entre otros.

Desarrollo de la Página Web

El desarrollo de una página web suele ser más sencillo de lo que se piensa, gracias a la amplia disponibilidad de plataformas que ofrecen códigos gratuitos y de paga (dependiendo de la complejidad o cantidad de código requerido), para su creación. Además, existen sitios que otros usuarios crearon donde nos brindan la variedad de códigos y sus significados, con el objetivo de facilitar aún más el proceso de creación.

Objetivo de la Práctica

El objetivo de esta práctica es desarrollar una página web sencilla que sirva como introducción a la tecnología, permitiendo a los estudiantes familiarizarse con herramientas útiles para su futuro académico y profesional. El propósito final es la creación de un portafolio web universitario propio.

Herramientas Utilizadas

Crearemos una página web utilizando HTML, el lenguaje básico de programación web, y emplearemos Visual Studio Code como herramienta principal. Esta plataforma versátil permite editar, depurar y compilar código de manera eficiente, además de facilitar la publicación final de la aplicación.

Pasos para Crear la Página Web

Paso 1

Crear una carpeta en tus documentos llamada “Mi primera página”. Abrir Visual Studio Code y seleccionar "Open File". Elegir la carpeta recién creada. Dentro de esa carpeta, crea un archivo con el nombre “index.html”. NOTA: El motivo por el cual se le asigna este nombre es debido a que los servidores al momento de cargar nuestro código para convertirlo en página ya reconocen el nombre como la página principal.

Paso 2

Agregar la estructura básica del HTML. Este código se ve reflejado de esta manera porque trabaja con etiquetas, ya que es un lenguaje de marcado (sirve para poner contenido). Abrir la extensión Live Server. La extensión es una ayuda para visualizar nuestros avances de manera eficaz.

Paso 3

Para visualizar nuestra página se agregarán nuevos códigos.

Paso 4

Agregar dentro de nuestra carpeta otra llamada “img” para descargar la imagen e ingresarla ahí. Copiar y pegar la imagen. NOTA: El motivo por el cual se le asigna este nombre es debido a que los servidores al momento de cargar nuestro código para convertirlo en página ya reconocen el nombre como la página principal.

Paso 5

Dar click a cualquier lugar del código y seleccionar “Open with Live Server” para visualizar nuestros arreglos.

Paso 6

Agregar los códigos que se visualizan en la imagen. Descargar otra imagen y agregarla a la carpeta e insertarla en href. Copiar index.html y pegarlo ahí mismo, posterior renombrarlo como Practica2. NOTA: El motivo por el cual se le asigna este nombre es debido a que los servidores al momento de cargar nuestro código para convertirlo en página ya reconocen el nombre como la página principal.

Paso 7

Modificar la Página2 con el fin de crear un hipervínculo de la página principal y la secundaria.

Paso 8

Buscar GitHub en la web y registrarse. NOTA: El motivo por el cual se le asigna este nombre es debido a que los servidores al momento de cargar nuestro código para convertirlo en página ya reconocen el nombre como la página principal.

Paso 9

Accede a tu cuenta de GitHub. Haz clic en el botón "New" para crear un nuevo repositorio. Asigna un nombre al repositorio de tu elección. No es necesario agregar una descripción. Asegúrate de que el repositorio sea público, ya que todas las páginas web son públicas. No modifiques ninguna otra configuración. Dale en “Crear repositorio”. Carpeta creada dentro de GitHub.

Paso 10

Darle en “Insights” y agregarle las carpetas.

Resultado
